Manual Verification Formula

You can verify this calculation manually using a calculator:

Note: Interest accumulates daily regardless of payment frequency
Payment frequency only affects how much interest is "due" when making payments

Step 1: Calculate daily interest rate
Daily Rate = Annual Rate ÷ 360
Daily Rate = {{ "%.2f"|format(loan.interest_rate * 100) }}% ÷ 360 = {{ "%.6f"|format(loan.interest_rate / 360) }}

Step 2: Calculate daily interest amount
Daily Interest = Principal × Daily Rate
Daily Interest = ₹{{ "%.2f"|format(loan.principal_amount) }} × {{ "%.6f"|format(loan.interest_rate / 360) }} = ₹{{ "%.2f"|format(loan.principal_amount * loan.interest_rate / 360) }}

Step 3: Calculate total interest for {{ days_active }} days
Total Interest = Daily Interest × Days
Total Interest = ₹{{ "%.2f"|format(loan.principal_amount * loan.interest_rate / 360) }} × {{ days_active }} = ₹{{ "%.2f"|format(loan.principal_amount * loan.interest_rate / 360 * days_active) }}

Step 4: Subtract verified payments
{% set verified_payments = payments | selectattr('payment.status', 'equalto', 'verified') | list %} {% set total_verified_calc = verified_payments | sum(attribute='payment.interest_amount') if verified_payments else 0 %} Accumulated Interest = Total Interest - Verified Payments
Accumulated Interest = ₹{{ "%.2f"|format(loan.principal_amount * loan.interest_rate / 360 * days_active) }} - ₹{{ "%.2f"|format(total_verified_calc) }} = ₹{{ "%.2f"|format(loan.principal_amount * loan.interest_rate / 360 * days_active - total_verified_calc) }}
