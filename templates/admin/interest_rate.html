How Interest Rates Work
Individual Loan Rates

Each loan now has its own specific interest rate. You can set different rates for different customers and loan types when creating or editing loans.

Key Features:
  • Flexible Rates: Set different interest rates for each loan
  • Easy Management: Change rates when creating or editing loans
  • Customer-Specific: Offer different rates based on customer risk profile
  • Loan-Type Specific: Different rates for personal, business, or emergency loans
Creating New Loans

When creating a new loan:

  1. Go to Create Loan page
  2. Enter the loan details
  3. Set the Annual Interest Rate (e.g., 21.00 for 21%)
  4. Choose payment frequency (Daily/Monthly)
  5. Save the loan
Create New Loan
Editing Existing Loans

To change an existing loan's interest rate:

  1. Go to All Loans page
  2. Click the Edit button (pencil icon)
  3. Modify the Annual Interest Rate
  4. Update other details if needed
  5. Save changes
